# Flutter UI Package

A comprehensive Flutter UI component library with customizable theming.

## Package Structure
```
app_ui/
├── lib/
│   ├── app_ui.dart
│   ├── theme/
│   │   ├── app_theme.dart
│   │   ├── theme_provider.dart
│   │   ├── theme_extension.dart
│   │   └── tokens/
│   │       ├── colors.dart
│   │       ├── typography.dart
│   │       ├── spacing.dart
│   │       ├── border_radius.dart
│   │       └── shadows.dart
│   ├── core/
│   │   ├── base_widget.dart
│   │   └── enums.dart
│   └── components/
│       ├── button/
│       │   ├── app_button.dart
│       │   └── button_enums.dart
│       ├── input/
│       ├── card/
│       └── ... (all other components)
└── test/
```
